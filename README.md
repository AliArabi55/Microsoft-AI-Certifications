## üìö Learning Tips

### For AI-900:

- Start with core AI concepts
- Focus on understanding Azure AI services
- Practice using the Azure Portal


### For AI-102:

- Ensure strong programming skills (Python or C\#)
- Emphasize hands-on practice
- Explore Azure AI Studio and Azure AI Foundry
- Delve into RAG solutions and fine-tuning
